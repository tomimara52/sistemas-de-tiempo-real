# Sistemas de tiempo real

Proyectos para correr en una STM32 NUCLEO-H533RE.


## Compilar
```
$ ./make.sh
```
actualizar ```make.sh``` para incluir en el path los ejecutables de stm32cubeide


## Debugear
```
$ ./stlink.sh
```
al igual que con ```make.sh```, actualizar ```stlink.sh``` para usar el stlink-gdbserver de stm32cubeide
